# Символы чудовищ

Описанные ниже символы помогут ведущему быстро определить наиболее подходящий образ действий для существа в бою. Кроме этого, их можно использовать, чтобы при создании сцены быстро найти существо для определённой роли. Если при наличии на странице нескольких блоков параметров слева указан лишь один символ, он относится ко всем этим существам.

## [Воитель]  
Воители прекрасно подходят для боя (вблизи, на дистанции или во всех ситуациях).

## [Заклинатель]  
Заклинатели в основном полагаются на заклинания и псевдозаклинания.

## [Специалист]  
Специалисты умело применяют всевозможные навыки.
